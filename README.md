# Eun Easy Uninstall/Linux上轻松卸载
你是否有想卸载一个软件却怎么也找不到它的软件包名？你是否因为卸载软件时被迫打开命令行而对Linux心生畏惧？现在这个问题彻底解决了，EasyUninstall有简单的图形化界面，软件图标和软件包名一一对应。只要认识软件图标，不用一句命令，就能在EasyUninstall里一键卸载并清除多余软件包。Linux新手必备软件！

### 功能
- 💻右键列表可自动使用“apt purge"彻底卸载软件


- 📦点击右下角按钮可自动使用“apt autoremove”移除多余依赖

- 🔔顶部可选展示软件类型（不启用复选框就只显示桌面能出现的软件）
## 安装和卸载
本软件仅支持Ubuntu/Debian系Linux

### 安装
下载好.deb包后直接双击或使用命令

      sudo dpkg -i easy-uninstall_1.0-1_amd64
      
### 卸载
删除/opt/easy-uninstall下所有文件或使用命令

      sudo apt purge easy-uninstall

# Eun Easy Uninstall/Uninstall software package on Linux easily
Have you ever wanted to uninstall software but couldn't find its package name? Are you intimidated by Linux because you were forced to open the command line when uninstalling software? Now that problem is completely solved! EasyUninstall has a simple graphical interface where software icons and package names are clearly matched. As long as you recognize the software icon, you can uninstall and remove unnecessary packages with a single click in EasyUninstall without using any commands. It's an essential tool for Linux beginners!

- 🔔Select the type of software to display（Only show software on desktop if you don't enable the cheakbox）
## Install and uninstall
This software only supports Ubuntu/Debian-based Linux distributions.

### Install
Double click .deb file or enter command.

      sudo dpkg -i easy-uninstall_1.0-1_amd64
      
### Uninstall
Delete all files in /opt/easy-uninstall or enter command

      sudo apt purge easy-uninstall
