# Eun Easy Uninstall/Linux上轻松卸载
你是否有想卸载一个软件却怎么也找不到它的软件包名？你是否因为卸载软件时被迫打开命令行而对Linux心生畏惧？现在这个问题彻底解决了，EasyUninstall有简单的图形化界面，软件图标和软件包名一一对应。只要认识软件图标，不用一句命令，就能在EasyUninstall里一键卸载并清除多余软件包，Linux新手必备软件！

### 功能
- 💻右键列表可自动使用“apt purge"彻底卸载软件


- 📦点击右下角按钮可自动使用“apt autoremove”移除多余依赖

- 🔔顶部可选展示软件类型（不启用复选框就只显示桌面能出现的软件）
## 安装
本软件仅支持Ubuntu/Debian系Linux

下载好.deb包后直接双击或使用命令

      sudo dpkg -i easy-uninstall_1.0-1_amd64
